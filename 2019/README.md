### 智慧穿戴</br>
##### 展項「智慧穿戴」主要負責燈光控制與互動裝置機電整合部份</br>
##### 第一次嘗試使用 Arduino 與 RaspberryPi 相互溝通
##### 按下按鈕時螢幕（RaspberryPi）會播放對應的影片，然後旁邊的櫥窗（Arduino）會照亮對應的實體物件
